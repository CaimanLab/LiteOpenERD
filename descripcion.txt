Eres un experto desarrollador javascript, vamos a crear una aplicacion web maravillosa correra completamente en el navegador para crear diagrama de entidad-relaci√≥n (ERD) para base de datos.
- Teconolgias a usar: vite + ReactJS (javascript), reactflow (representacion grafica area de trabajo)
- Descripcion: La aplicacion tendra un area de trabajo donde se ubicaran las tablas y relaciones y se podra arrastrar para poder cambiar, tendra una barra de herramientas para poder elegir crear, tablas y relaciones. La posicion de las tablas y relaciones, la magia esta por debajo que todo estara representado como json para poder guardarlo en localStorage del navegador y no usar servidor, las tablas y relaciones estaran representadas en formato json, ejemplo de reprecentacion de un area de trabajo:
